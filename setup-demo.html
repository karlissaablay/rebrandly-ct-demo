Conversion Tracking Setup Demo

See how easy it is to install Rebrandly Conversion Tracking on your website and start measuring what matters.

Enable Conversion Tracking on Your Links

Toggle tracking on for individual links or set it as the default for your entire workspace.

Rebrandly — Edit Link
rebrand.ly / metricflow-demo
https://karlissaablay.github.io/rebrandly-ct-demo/
Disabled

When enabled, a ?rbly_click_id= parameter will be appended to the destination URL on redirect.

How it works

1
Toggle tracking ON

Enable conversion tracking for any link in your Rebrandly dashboard. Try clicking the toggle.

2
Click ID is appended automatically

When someone clicks your branded link, Rebrandly appends ?rbly_click_id=abc123 to the destination URL.

3
Set attribution window

Choose how long after a click a conversion can be attributed. Default is 30 days.

Install the Tracking Snippet

Add one line of code to your website's header. Works with any CMS or static site.

Website Settings — Header Scripts

Paste scripts or code snippets that should appear in the <head> section of every page on your site.
